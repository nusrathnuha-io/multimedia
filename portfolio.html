Project 4: Mobile Application for Library Management

Project 5: Mobile Application for Garbage Collection (BinBuddy)

BinBuddy is a mobile app designed to optimize garbage collection management. It provides features such as scheduling pickups, tracking bin levels, and notifying users about waste collection times, contributing to a cleaner environment and efficient city services.

Project 6: Indoor Wi-Fi Router Placement Optimization

This project focuses on optimizing the placement of indoor Wi-Fi routers to ensure maximum signal strength and coverage. It takes into account the layout of the space and potential obstacles to provide an optimal router arrangement for better connectivity.
